# Vlast Community Adapter

Адаптер-сайт для сообщества Vlast (пользователь: @swatdc).

## Структура проекта

- `index.html` - основная структура сайта
- `style.css` - стили и анимации
- `script.js` - логика навигации и взаимодействия

## Функциональность

1. **Вступительная анимация** - 5-секундное приветствие с анимацией текста
2. **Главное меню** - 4 основные категории:
   - Проекты
   - Соц сети
   - Менеджеры и дизайнеры
   - Услуги
3. **Вложенные экраны** - многоуровневая навигация с кнопкой "Назад"
4. **Внешние ссылки** - все кнопки-ссылки открываются в новой вкладке

## Особенности дизайна

- Анимированный градиентный фон (от черного к красному)
- Современные анимации переходов
- Адаптивный дизайн для мобильных устройств
- Иконки Font Awesome
- Google Fonts (Orbitron, Russo One, Montserrat)

## Как редактировать

### Добавить новый экран:
1. В `index.html` создать новый `<div class="screen">`
2. Добавить обработку в `script.js` в объект `screens`
3. Создать кнопку для перехода на этот экран

### Изменить ссылки:
Все ссылки находятся в `index.html` в атрибутах `href` у соответствующих кнопок.

### Изменить цвета:
Основные цвета заданы в `style.css`:
- Основной красный: `#ff0000`
- Темные оттенки: `#330000`, `#4d0000`
- Градиенты в `.menu-btn`

### Добавить анимации:
CSS-анимации находятся в конце файла `style.css`.
